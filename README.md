# 专升本报名系统

软件实训，实操项目。

## 采用技术

- Vue

- Element-ui

- axios

## 安装

```
git clone git@github.com:popring/registration-system.git

cd registration-system

// 可以使用 npm 替代
yarn

// 运行
yarn serve

```
